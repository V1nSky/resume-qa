Пример баг-репорта

Bug · Medium

Publish Now отображается для уже опубликованного элемента очереди

В интерфейсе Publishing Queue кнопка Publish Now отображалась для контента со статусом Published. Это могло запутать пользователя и привести к повторной попытке публикации.

Шаги

  1. Открыть ZodiacOps
  2. Перейти в Publishing Queue
  3. Найти элемент со статусом Published
  4. Проверить доступные действия

Результат

  • Факт: кнопка Publish Now отображается.
  • Ожидание: для Published элемента кнопки быть не должно.
  • Priority: High
  • Environment: macOS, Chrome, React/Vite, FastAPI
